# 前言

# 題目

有一個封閉的長方體撞球室（所有內壁皆為完美反射鏡面）位於第一卦限，其大小為：

- **長 (X 軸方向) $L = 12$**
  
- **寬 (Y 軸方向) $W = 10$**
  
- **高 (Z 軸方向) $H = 15$**

我們將左下角的底面頂點設為原點 $(0, 0, 0)$。現在有一顆質點小球從原點出發，以特定的方向向量 **$\vec{v} = (3, 4, 5)$** 射出。當球撞擊到任何牆面（包含天花板與地板）時，皆符合完全彈性碰撞的反射定律。

請試著計算並回答以下三個問題：

## 1. 終局之點

這顆球在空間中不斷彈跳，最終「第一次」落入長方體的某一個頂點（角落）時，**該頂點的真實空間座標 $(x, y, z)$ 為何？**

## 2. 碰撞計數

在小球從出發到「剛好掉進該頂點」的整個運動過程中，**它總共與牆面經歷了幾次碰撞？**（不包含最後落入頂點的那一下）

> （注意：這題是三維空間，請小心思考當球「同時撞到兩面牆（即撞到長方體的邊框）」時，物理上只算一次碰撞）

## 3. 魔王加分題：稜線擦撞

承接上題，小球在行進過程中，是否曾經「完美擊中長方體的某一條稜（邊框）」**？如果是，請找出**這是它的第幾次碰撞？。

# 解答

首先假設鏡面展開後在$x,y,z$方向各變成了$p,q,r$倍，則$\frac{12p}{3}=\frac{10q}{4}=\frac{15r}{5}$，且$\gcd(p,q,r)=1$，解得$(p,q,r)=(15,24,20)$。偶數時會打回同側，奇數會打到對側，因此會落入角落$(12,0,0)$。

欲計算反彈幾次，可將問題看成由單位方格組成的$15\times24\times20$長方體的對角線穿過的面數，扣除穿過的稜數。$\gcd(15,24)=3, \gcd(15,20)=5,\gcd(24,20)=4$，故所求為$(15-1)+(24-1)+(20-1)-(3-1)-(5-1)-(4-1)+(1-1)=47$次。

稜線擦撞可先把長方體投影在$xy,yx,zy$任一平面上，即可化歸為矩形碰撞問題。而由運動獨立型可知各方向的稜碰撞互相獨立。

首先看$xy$平面，此情形算出的稜線碰撞將會是撞到平行$z$軸的稜。假設鏡面展開後$x,y$方向分別變成了$\alpha,\beta$倍，則$\frac{12\alpha}{3}=\frac{10\beta}{4},\gcd(\alpha,\beta)=1$，解得$(\alpha,\beta)=(5,8)$(不難想到，其實，$\frac{\alpha}{\beta}$即$\frac{p}{q}$的最簡分數)，故每歷經$(5-1)+(8-1)=11$次撞擊垂直$xy$平面的平面後，會撞到平行$z$軸的稜，且共撞擊$\gcd(p,q)-1=2$次(此計算以排除結束時那個頂點)(亦可用$\frac{(15-1)+(24-1)-2k}{11}=k+1$，解得$k=2$。左式分子減$2k$是因為碰到稜在平面投影的模型中不會算入碰撞次數，如此每次碰稜都會少$2$次碰撞次數($x$平面和$y$平面的))。此系列第一次撞擊的稜為$x-12=y=0$。接著由碰撞軌跡的可逆性之第二次撞擊$x=y=0$，如此來回。

類似可計算得故每歷經$5$次撞擊垂直$xz$平面的平面後，會來回撞$x-10=z=0$和$x=z=0$共$4$次

每歷經$9$次撞擊垂直$yz$平面的平面後，會來回撞$y=z-15=0$和$y=z=0$(不計始終)共$3$次

由於$v=(3,4,5)$且$(L,W,H)=(12,10,15)$，每次撞擊垂直$x$、$y$、$z$軸的平面的時間間隔分別為$8$、$5$、$6$個單位時間。將撞擊垂直$x,y,z$軸的平面分別記作$X,Y,Z$，每個方向撞擊時間如下：

當某個時間點恰好是$[8,5]=40$的倍數，則會撞擊平行$z$軸的稜，餘同理。而當$t=[8,5,6]=120$時即代表撞擊結束(碰到頂點)。因此撞擊稜線的時間點分別為

$XY:t=40,80$

$XZ:t=24,48,72,96$

$YZ:t=30,60,90$

各方向的面撞擊(排除撞稜)時間如下：

$X:t=8,16,32,56,64,88,104,112$

$Y:t=5,10,15,20,25,35,45,50,55,65,70,$

               $75,85,95,100,105,110,115$

$Z:t=6,12,18,36,42,54,66,78,84,102,108,114$

因此撞稜分別發生在第$10, 12, 16, 19, 24, 29, 32, 36, 38$次碰撞。
