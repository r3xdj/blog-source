# 前言

# 數字狼人殺

## 題目敘述

小明選取了兩個正整數$x,y$，滿足$1<x<y<10$，他將$x+y$告訴甲、$xy$告訴乙、$x-y$告訴丙。已知甲、乙、丙皆有可能說實話或謊話，但說實話者永遠說實話，說謊話者永遠說謊話，且在開始對話前，甲、乙、丙三人互相對彼此說實話或謊話的情形一無所知。假設甲、乙、丙都是絕頂聰明的，能運用所有現有知識做出盡可能多的推論，現在，他們開始以下對話：

甲：我不知道這兩個數是什麼，但我知道這兩個數中一定有奇數
乙：我不知道這兩個數是什麼
丙：我不知道這個數是什麼，不過甲在說謊
甲：乙不知道這兩個數是什麼
乙：的確，我不知道
丙：甲不知道這兩個數是多少，而且乙在說謊
甲：我不完全知道乙和丙是否說謊的全部實情
乙：我手上的數字($xy$)是偶數
丙：我現在知道這兩個數是什麼

(註：三人都是在確認自己說的話一定是實話或謊話才會講，以避免說出不小心說出不符合自己人設的話；另外，要整句話皆為真才是實話)
請問：

(以下2~5題皆承接1.，即假設真正的$(x,y)$即是使$x+y$達最小的那組)

1. $(x,y)$ 的所有可能為何？使得$x+y$最小的那組$(x,y)$為何？
  
2. 丙說完他的第二句話後，據甲所知所有他認為有可能的$(x,y)$所組成的最小集合為何？
  
3. 甲、乙、丙三人知道$(x,y)$確切值的順序為何？分別在何時就知道了？
  
4. 原對話刪除其中$k$句，且確保三人的邏輯嚴謹與合理，若欲使丙仍能判斷出$(x,y)$的確切數值，試求出$k$的最大值，以及具體刪除哪幾句？
  
5. 乙和丙分別在什麼時後開始知道「甲知道乙、丙誰說謊，誰說實話」？在乙或丙知道的當下，甲知道乙或丙知道自己知道這件事嗎？若否，甲又是在什麼時後才開始知道，乙或丙早在當時就知道自己知道了呢？

> 為了以下論述方便，我們使用`說話人-全部句數中第幾句`對句子進行編號，譬如`乙-5`為「的確，我不知道」；另外，以T代表說實話者，F代表說謊者；再者，若論述提及「某人不知道」，未註明時均指他「不知道$x,y$兩數的確切數值」。

為了以下論述方便，我們使用`說話人-全部句數中第幾句`對句子進行編號，譬如`乙-5`為「的確，我不知道」；另外，以T代表說實話者，F代表說謊者；再者，若以下論述提及「某人不知道」，未註明時均指他「不知道$x,y$兩數的確切數值」。以下開始進行推論：

- 甲知道$x+y$中必有奇數，若且惟若$x+y$為奇數，因為此時$x,y$必為一奇一偶，且若$x+y$為偶數，甲將不能確定$x,y$是兩奇或兩偶
  
- $x+y$和$x-y$的奇偶性相同
  
- 丙-3：「我不知道這個數是什麼，不過甲在說謊」
  
  - 若丙為T $\Rightarrow$ 丙不知道($\Rightarrow$ $x-y\neq-7$)且甲為F，而且<font color=pink>丙知道甲為F</font>。根據上表，我們知道丙除非是拿到$-7$，否則不可能在一開始就能推論出甲知道與否，因此丙僅能對「$x+y$必有奇數」做出否定 $\Rightarrow$ $x+y$和$x-y$均為偶數
    
  - 若丙為F $\Rightarrow$ 丙知道($\Rightarrow$ $x-y=-7$)或甲為T，不過若丙知道，則查表知甲確實不知道，且丙知道代表$x-y=-7$，進而$x+y$為奇數，從而甲亦說實話。因此，無論如何甲為T。$\Rightarrow$ $x+y,x-y$為奇數，且$x+y\notin\{5,6,16,17\}$
    
  - 由以上兩點，我們得知：甲為T $\Leftrightarrow$ 丙為F $\Leftrightarrow$ $x,y$一奇一偶，且由於這些推理甲、丙都會做出，又他們知道自己是否說謊，因此他們互相知道對方是否說謊(在ROUND 1結束後)
    
- 丙-6：「甲不知道這兩個數是多少，而且乙在說謊」
  
  - 若丙為T，則有
    
    1. 甲為F，且丙知道這件事
      
    2. 甲知道丙為T
      
    3. $x+y,x-y$為偶數
      
    4. 丙知道「甲不知道這兩個數是多少」
      
    5. 丙知道「乙在說謊($\Rightarrow$ 乙知道 $\Rightarrow$ $xy\notin\{12,18,24\}$)」
      
    
    - 由1，甲-4為假，因而「乙知道這兩個數是多少」，從而乙亦為F，且甲知道這件事，因而甲由手上的$x+y$可知$xy\notin\{12,18,24\}$。我們將所有可能根據3進行篩選，滿足$xy\in\{12,18,24\}$的組合只剩$(x,y)=(2,6)\vee(4,6)$，由於要能由$x+y$排除這種情形的可能性，從而$x+y\neq8\vee10$。查表可知$(x,y)\in\{(2,4),(3,9),(4,8),(5,7),(5,9),(6,8),(7,9)\}$

      $(x,y)\in\{(3,9),(4,8),(5,9)\}$
      
    - 此時有(甲,乙,丙)=(F,F,T)
      
  - 若丙為F，則由甲-4為真，乙-5亦為真，從而(甲,乙,丙)=(T,T,F)。且由於甲、乙、丙都能做出相應推理(他們擁有的資訊甚至還比我們多，我們在資訊受限時能做出的推理他們自然能做出)，因此他們三人也都知道彼此的說謊或說實話情形。從而甲-7：「我不完全知道乙和丙是否說謊的全部實情」為謊話，但這與甲說實話矛盾，從而此情形不可能發生
    
  - 綜合以上分析，我們得出
    (甲,乙,丙)=(F,F,T)，且$(x,y)\in\{(3,9),(4,8),(5,9)\}$，且可驗證此人設組合與甲-7並不矛盾
